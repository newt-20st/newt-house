イベントデータ

APIエンドポイント

  • https://script.google.com/macros/s/AKfycbwe2pjvkmHz2FYhdNxf96QvDEgDtGgk4_7mmyhFNLLIuw0O0vjDb2_aUVFv_qTlyu6M/exec?type=event

フィールド

no Int イベントナンバー
event-name String イベント名
start String イベント開始時間
end String イベント終了時間
time String イベントの長さ
event-type String イベントの形式
member List イベントサポートメンバー
unit String 担当ユニット。箱イベではない場合は-が返される
music String イベント楽曲。箱イベではない場合は-が返される
bunner String バナーのメンバー名
type String イベントタイプ
gacha String 同時に開催されるガチャの名前
user-count Int イベントに参加した総ユーザー数
user-count-source String user-countのデータソース

データ更新は不定期です。
