SHIP-NOTIFY

最終更新: 2021-01-26

これは何？

SHIP ( Sakae Higashi Internet Program : SHIPトップ ) の更新を取得し、LINE公式アカウントから通知するシステム( bot )です。友だち追加は下のボタンかQRコードからできます。

現在は5時、12時、17時、22時に取得しています。

仕様

制約など

  • LINE Messaging APIのフリープランの制約上1ヶ月に送れるプッシュメッセージ・ブロードキャストメッセージは1000通までです。自発的に何も送ってこなくなったなと思ったらそういうことなんだなと思っておいてください。ただし、リプライメッセージはカウント外なので !connectionコマンドや!studyコマンドを送れば最後にSHIPから取得した時のデータをみることができます。
  • このbotは中学のページのみ取得しています。高校のページのデータは取得していません。
  • このbotで取得できる更新は「連絡事項」や「学習教材」トップの新着情報の更新のみです。すでに上がっているファイルがあげ直したりされたかどうかなどについては取得していません。
  • メッセージを送信したタイミングによってはレスポンスに数十秒ほどかかる場合があります。
  • 複数人のトークルームやグループチャットでの動作は検証しておりません。

コマンド一覧

  • !about

    このbotについての情報を表示。下のリッチメニューからも送信可能

  • !command

    コマンド一覧を表示

  • !settings

    個人設定のコマンドを表示

  • !issue 不具合内容

    不具合報告用。機能要望なども受付

  • !when

    現在のページを取得するタイミングを表示

  • !connection

    「連絡事項」ページの最終取得時のデータを表示。下のリッチメニューからも送信可能

  • !study

    「学習教材」ページの最終取得時のデータを表示。下のリッチメニューからも送信可能

  • !notify-all

    すべての通知をオンにします。SHIPを取得するたびに通知します。その他、細かな報告などについても通知することがあります

  • !notify-middle

    通知の量を減らします。SHIPに更新がなかった際の通知は3日に一度ほどになります。大きな機能追加・改善などのプッシュメッセージの送信なども行います

  • !notify-few

    通知を最低限にします。SHIPに更新があったときのみ通知します
