SHIP-NOTIFY

最終更新: 2021-01-18

これは何？

SHIP ( Sakae Higashi Internet Program : SHIPトップ ) の更新を取得し、LINE公式アカウントから通知するシステム( bot )です。友だち追加は下のボタンかQRコードからできます。

仕様

機能の要望などがあれば!issueコマンドからお願いします。

制約など

  • LINE Messaging APIのフリープランの制約上1ヶ月に送れるプッシュメッセージ・ブロードキャストメッセージは1000通までです。自発的に何も送ってこなくなったなぁと思ったらそういうことだと思っておいてください。ただし、リプライメッセージはカウント外なので !connectionコマンドや!studyコマンドを送れば最後にSHIPから取得した時のデータをみることができます。
  • このbotは中学のページのみ取得しています。高校のデータは取得していません(今後気が向いたら追加するかも)

コマンド一覧

  • !about

    このbotについての情報を表示。下のリッチメニューからも送信可能

  • !command

    コマンド一覧を表示

  • !settings

    個人設定のコマンドを表示

  • !issue 不具合内容

    不具合報告用。機能要望なども受付

  • !when

    現在のページを取得するタイミングを表示

  • !connection

    「連絡事項」ページの最終取得時のデータを表示。下のリッチメニューからも送信可能

  • !connection

    「学習教材」ページの最終取得時のデータを表示。下のリッチメニューからも送信可能

利用規約とプライバシーポリシー的なもの

  • 当botはユーザーから運用にあたって必要なデータ(LINEユーザーIDなど)を取得します。
  • 運用者は当botを利用したことによって生じたいかなる責任も負いません。すべて自己責任でご利用ください。
  • 当botは栄東中学校に所属する方のみ利用可能とします。運用者はユーザーが栄東中学校に所属していないと判断した場合、予告なくサービスの提供を停止します。
  • 当botは予告なく(目に見える形での通知無しで)運用を中止することがあります。予めご了承ください。
  • 当botは取得する個人情報を最低限に留め、また当botの機能改善のためのみに使うものとし、第3者へ情報を提供することは一切しません。
  • この利用規約およびプライバシーポリシーはユーザーが当botをフォロー(友だち追加)している間に有効なものとします。ユーザーは当botをアンフォロー(ブロック)することによってサービスの利用を停止することができ、またその時運営者はアンフォローを行ったユーザーの個人情報の取得を行うことができなくなります。
